Overview

Repute is a multipurpose website template based on Bootstrap framework. Save your time, create high quality website faster either for business, corporate, agency or even personal. Repute provides you many options and features to fulfill your website requirements. With on-going development, Repute will have additional options and features in every release update, for free. It is easy to customize Repute since it's built with developer-friendly and clean-coded in mind. Even better, we provide you extensive documentation to get started or dig more. If you use Sass and Compass, you have a good news because Repute is built with Sass and Compass. If you have any problem using the theme, do not hesitate to reach our support for free.

Important Notes

Working With Contact Form

Contact form is working with AJAX call and later send an email once the visitor submitted the message. To change the destination email address please open php/contact.php and find $emailTo variable as below.

Provide your email address on $emailTo and actually you can change how the email content will look like by changing the $subject and $body value. Also you can change the success message for the visitor if the message has been sent by changing $message value.

Working With Newsletter

In footer section of the theme, you will find newsletter form which work with mailchimp. You can find the AJAX call script for newsletter at assets/js/repute-scripts.js and find "newsletter". As you can see, the script calls php/mailchimp.php to do the backend process. That's the file where we put some newsletter parameters setting. Please open php/mailchimp.php and provide API key and list ID which you can get them from your MailChimp account.

Note: In order to make the newsletter form works, please make sure the theme is placed in web server with PHP environment, otherwise it doesn't work (i.e. opening the template as static file instead on localhost).

One Page

One-page HTML is basically derived from main theme, only it has 1-level menu with scrolling navigation and just consists of one page for the whole content. Consequently, the CSS, Javascript, images and others are taken from the same assets/ directory as main theme. Exception is made if only the CSS or Javasript needed is specific to the one-page only, for example at one-page/assets/js/ directory. However, this is a way to simplified things and avoid too many code/asset files duplication. In your one-page project, you may be just copy the asset files necessary from main theme and work in separate directory.

Navigation Bar

Beside default navigation bar, the theme also has 3 other types of navigation bar, i.e. Fixed Navbar, Auto Hiding, Fixed & Shrink. To see how these features in action, please visit our theme demo, open demo panel in left side and click at navigation bar type, then scroll the page.

Fixed Navbar

To apply fixed navbar, add .navbar-fixed-top to .navbar class. You might want to adjust padding-top on .wrapper since applying fixed navbar will make its position as fixed then make hero unit move up.

Auto Hiding

Auto hiding will make the navigation bar disappear as you scroll down the page to give more space, then it will appear again as you scroll up. This feature use Bootstrap Auto-Hiding Navbar, so make sure you have included plugin file.

Fixed & Shrink

This feature is useful if you have a relatively tall navigation bar and but want it to be fixed, for example navigation bar with top links and search form and main navigation links below it. Perhaps tall navbar with fixed position is not good idea since it will take more screen, so we shrink and hide the less usefull elements, in this case, the top area. You can find the demo at index-v3-fixedshrink.html. To apply this feature add .navbar-fixed-top and .shrinkable classes to .navbar, see below.
